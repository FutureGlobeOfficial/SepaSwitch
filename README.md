# Sepa Switch

## Dependencies

This extension only works with the extension [Excel Viewer](https://marketplace.visualstudio.com/items?itemName=GrapeCity.gc-excelviewer). Make sure to have it installed and active

## How to use

There are two ways you can switch the separator:

- Open up VS Code Command Palette (<span style="color: dodgerblue">Ctrl+Shift+P </span> or <span style="color: dodgerblue">Cmd+Shift+P</span>) and type in the command: <span style="color: dodgerblue">Change Separator</span>

- Use the keyboard shortcut (<span style="color: dodgerblue">Ctrl+Alt+L </span> or <span style="color: dodgerblue">Cmd+Alt+L</span>) to toggle the separator input field directly.
